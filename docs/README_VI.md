# 🎬 Video Speed Controller

Một script giúp bạn **tăng/giảm tốc độ phát video** trên bất kỳ trang web nào chỉ bằng phím tắt.

## ✨ Tính năng

- ⏩ Tăng/Giảm tốc độ phát video bằng phím `D` / `S`
- 🔁 Đặt lại tốc độ về mặc định (1.0x) bằng phím `R`
- 💾 **Tự động lưu tốc độ hiện tại** cho từng website riêng biệt
- 📺 **Hiển thị tốc độ video** dưới dạng overlay mờ đẹp mắt
- 🛠️ **Cấu hình tùy chỉnh**: Thay đổi bước tăng/giảm, phím tắt, và ngôn ngữ qua menu cấu hình
- 🔄 **Đặt lại cấu hình**: Khôi phục tất cả cài đặt về mặc định
- 🌍 **Hỗ trợ đa ngôn ngữ**: Tiếng Anh và Tiếng Việt
- ⚙️ Hỗ trợ tất cả các website có sử dụng tag `<video>`, bao gồm YouTube, Facebook...

## ⌨️ Phím tắt

| Phím  | Chức năng              |
|-------|------------------------|
| `D`   | Tăng tốc độ +0.5x      |
| `S`   | Giảm tốc độ -0.5x      |
| `R`   | Đặt lại tốc độ = 1.0x  |

> **Lưu ý**: Không hoạt động nếu bạn đang gõ trong input/textarea.

## 🧠 Lưu tốc độ thông minh

- Mỗi khi bạn thay đổi tốc độ, script sẽ **ghi nhớ tốc độ cho từng website**.
- Khi bạn **tải lại trang**, tốc độ đã lưu sẽ **tự động được áp dụng** cho tất cả video trên trang.

## 🖼 Overlay trên video

- Hiển thị tốc độ hiện tại ở góc trên bên trái video.
- Tự động ẩn sau 1 giây để không làm phiền trải nghiệm xem.

## ⚙️ Menu cấu hình

- Nhấn chuột phải vào icon của userscript manager (ví dụ: Tampermonkey) để truy cập menu
- **Mở cấu hình**: Tùy chỉnh bước tăng/giảm tốc độ (delta), phím tắt cho tăng/giảm/đặt lại, và ngôn ngữ giao diện
- **Đặt lại toàn bộ cấu hình**: Khôi phục tất cả cài đặt về giá trị mặc định

## 🛠 Cài đặt

1. Cài extension [ScriptCat](https://docs.scriptcat.org/) hoặc [Tampermonkey](https://www.tampermonkey.net/) cho trình duyệt.
2. Click để cài đặt [script.user.js](https://raw.githubusercontent.com/dongcodebmt/video-speed-controller/main/script.user.js)
